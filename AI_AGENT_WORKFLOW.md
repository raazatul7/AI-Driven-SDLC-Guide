# AI_AGENT_WORKFLOW.md
## Multi-Agent Simulation Workflow

---

# Stage 1: Product Manager

- Draft PRD
- Define scope
- Define success metrics

---

# Stage 2: Architect

- Convert PRD to TRD
- Define system architecture
- Define modules
- Define API standards

---

# Stage 3: Security Engineer

- Audit architecture
- Define encryption
- Define compliance requirements

---

# Stage 4: DevOps Engineer

- Design CI/CD
- Define environments
- Define container strategy
- Define scaling strategy

---

# Stage 5: Code Reviewer

- Validate modularity
- Validate testability
- Validate maintainability

---

# Stage 6: Scalability Engineer

- Stress test assumptions
- Identify bottlenecks
- Define scaling roadmap
