# QUALITY_GATE_CHECKLIST.md

---

# PRD Approval Checklist

- Problem defined
- Users defined
- Roles defined
- Features scoped
- Edge cases listed
- Success metrics defined

---

# TRD Approval Checklist

- Tech stack finalized
- Modules separated
- APIs defined
- Database relationships defined
- Security defined
- Performance targets defined

---

# Pre-Development Checklist

- Architecture approved
- ADRs written
- Security audit complete
- CI/CD defined

---

# Pre-Release Checklist

- Tests passing
- No critical bugs
- Load testing done
- Monitoring active
- Backup verified
- Rollback tested
